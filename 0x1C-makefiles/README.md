# 0x1C. C - Makefiles

## Description
This repository contains solutions for the tasks related to Makefiles in the C programming language. The tasks are part of the Holberton School curriculum.

## Requirements
- Linux OS
- GCC compiler

## Installation
To use the files in this repository, you can clone the repository using the following command:
git clone https://github.com/youssefberrk/0x1C-makefiles.git

## Usage
Each task directory contains a `Makefile` and additional source files necessary to build the programs.

To compile a specific task, navigate to the task directory and run the `make` command:

cd task_0x00
make


To clean up object files and executable files generated during compilation, run:
make clean

To clean up object files, executable files, and dependency files, run:
make re

## Files
- `0-Makefile`: Basic Makefile to compile a C file named `holberton`.
- `1-Makefile`: Makefile with variables and implicit rules.
- `2-Makefile`: Makefile with additional rules.
- `3-Makefile`: Makefile with additional rules and clean functionality.
- `4-Makefile`: Makefile with all variables.
- `5-island_perimeter.py`: Python script to calculate the perimeter of an island described by a grid.
- `README.md`: This README file.
